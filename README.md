# AndroidVision

General purpose Pytorch inference app, comprises a common Pytorch base framework (Setting up camera
, permissions, loading modules and constructing Android CameraX usecases ) for
 various
 computer
 vision
 tasks:

- Classification
- Segmentation
- Detection
